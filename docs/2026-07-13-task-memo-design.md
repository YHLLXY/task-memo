# 极简工作任务备忘录 — 设计文档

> 2026-07-13 | 独立项目 | PWA 打包

## 项目定位

极简化个人任务备忘录，移动端 PWA，可添加到手机主屏幕使用。纯前端，localStorage 存储，零依赖，零构建工具。

## 技术栈

- 纯 HTML/CSS/JS（单文件 `index.html`）
- localStorage 持久化
- PWA（manifest.json + Service Worker）
- 移动端优先，CSS Grid + Flexbox

## 数据模型

```typescript
interface SubTask {
  id: string;           // 唯一标识（时间戳+随机数）
  content: string;      // 子任务文字
  completed: boolean;   // 是否完成
}

interface Task {
  id: string;
  content: string;
  priority: 'high' | 'medium' | 'low';
  deadline?: string;        // ISO 字符串，可选
  pinned: boolean;          // 钉选置顶
  sub_tasks: SubTask[];
  created_at: string;       // ISO
  date: string;             // YYYY-MM-DD，所属日期
  completed: boolean;
  completed_at?: string;    // ISO
  order: number;            // 排序序号（箭头调整用）
}
```

## 功能清单

| # | 功能 | 交互 |
|:--:|------|------|
| 1 | 添加任务 | 底部输入框 + 优先级按钮 🔴🟡🟢 + 截止时间选择（可选）→ 回车或点击添加 |
| 2 | 任务列表 | 未完成在上（钉选优先 → 逾期优先 → 优先级 → 创建时间），已完成在下 |
| 3 | 完成确认 | 点击 ⭕ → 弹窗"确认完成？"→ 确认后下沉到已完成区 |
| 4 | 撤销完成 | 点击已完成任务的 ✅ → 恢复为未完成 |
| 5 | 内联编辑 | 点击任务文字 → 输入框替换文字 → 回车保存 |
| 6 | 钉选置顶 | 长按任务 → 弹出操作菜单 → "钉选/取消钉选" |
| 7 | 子任务 | 每个任务下可添加小步骤，独立复选框勾选 |
| 8 | 排序调整 | ↑↓ 箭头按钮调整未完成任务顺序 |
| 9 | 进度条 | 顶部完成率进度条 + "待完成 N · 逾期 X · 已完成 M" |
| 10 | 折叠已完成 | 已完成区域一键折叠/展开 |
| 11 | 删除 | 长按任务 → 弹出操作菜单 → 确认删除 |
| 12 | 清除已完成 | 底部"清除已完成"文字按钮，一键删除所有已完成 |
| 13 | 深色模式 | 🌓 按钮切换，默认跟随系统 `prefers-color-scheme` |
| 14 | 按天刷新 | 新的一天开始，列表只显示当天任务 |
| 15 | 30天清理 | 每次加载自动删除 30 天前的已完成任务 |
| 16 | PWA | manifest.json + sw.js，可添加到主屏幕全屏运行 |
